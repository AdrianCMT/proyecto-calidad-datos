# Analisis de Tendencias de Videojuegos

Proyecto de analisis exploratorio con Pandas y Numpy para estudiar tendencias en videojuegos a partir de datos historicos.

## Estructura del proyecto

- `notebooks/`: notebook principal del analisis.
- `data/`: dataset en formato CSV.
- `reports/`: informe interpretativo en TXT y DOCX.
- `requirements.txt`: dependencias del proyecto.

## Archivos principales

- `notebooks/analisis_tendencias_videojuegos.ipynb`
- `data/raw_data.csv`
- `reports/informe_interpretativo_videojuegos.txt`
- `reports/informe_interpretativo_videojuegos.docx`

## Como ejecutar

1. Crear entorno virtual:

```bash
python -m venv .venv
```

2. Activar entorno (PowerShell):

```bash
.\.venv\Scripts\Activate.ps1
```

3. Instalar dependencias:

```bash
pip install -r requirements.txt
```

4. Abrir y ejecutar el notebook:

```bash
jupyter notebook notebooks/analisis_tendencias_videojuegos.ipynb
```
